API Basejump: URL Shortener Microservice

User stories:
    1) I can pass a url or digit as a parameter, and it will check to see whether that value contains either the digit or a url (example: https://www.google.com)
    2) If its a url, output a shortened URL in the JSON response.
    3) If its a digit, redirect to save page with digit

Example usage:

https://site/https://www.google.com
https://site/1450137600

Example output:

{ "original_url":"http://foo.com:80", "short_url":"https://little-url.herokuapp.com/8170" }
